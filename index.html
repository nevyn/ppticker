ppticker av nevyn

Ett app för att följa hur många medlemmar Piratpartiet har JUST NU!
 

Datan kommer härifrån tror jag: http://lekstuga.piratpartiet.se/pp-ticker/

Installera

  • 1. Ladda ner
  • 2. Packa upp genom att dubbelklicka
  • 3. Dra till Applications eller kör från där det ligger

Källkod

Du kan tanka källkoden packat som zip eller tar.

Du kan också klona Git-repositoryt genom att skriva: 

$ git clone git://github.com/nevyn/ppticker
